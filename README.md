# Currency-Converter


The Currency Converter is a web-based application designed to convert one currency into another using current exchange rates. The system allows users to select the source currency and the target currency, enter an amount, and instantly view the converted value. The application performs real-time calculations by applying the latest exchange rate data, ensuring accurate and up-to-date conversions.

The project is built using standard web technologies such as HTML, CSS, and JavaScript. The user interface provides input fields for the amount and currency selection, while the underlying logic processes the conversion through predefined or **API-based exchange rates**. When a user enters a value and selects the desired currencies, the application calculates the equivalent amount and displays the result dynamically without requiring a page reload.

This project demonstrates concepts such as user input handling, DOM manipulation, event-driven programming, and integration of external exchange rate data. The system is useful for travelers, online shoppers, and businesses that frequently deal with multiple currencies. By simplifying the conversion process and presenting results instantly, the Currency Converter improves efficiency and usability for users needing quick financial calculations.
